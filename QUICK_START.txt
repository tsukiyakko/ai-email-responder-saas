STEP 1: GET API KEYS (15 minutes)
================================================================================

Clerk Auth (Free):
  → https://clerk.com
  → Create app → Copy NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY & CLERK_SECRET_KEY

Google Gemini (Free):
  → https://aistudio.google.com
  → Create API key → Copy GEMINI_API_KEY

Upstash Redis (Free tier):
  → https://upstash.com
  → Create database → Copy UPSTASH_REDIS_REST_URL & UPSTASH_REDIS_REST_TOKEN

Stripe (Free):
  → https://stripe.com
  → Create product: "Standard" $5/month recurring
  → Get STRIPE_PRICE_ID, STRIPE_SECRET_KEY, NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY
  → Add webhook to /api/stripe/webhook → Get STRIPE_WEBHOOK_SECRET

================================================================================
STEP 2: CONFIGURE ENVIRONMENT (2 minutes)
================================================================================

cd /path/to/ai-email-responder-saas
cp .env.local.example .env.local
nano .env.local  # or use your editor

Paste all API keys from Step 1:
  NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
  CLERK_SECRET_KEY=sk_test_...
  GEMINI_API_KEY=...
  STRIPE_SECRET_KEY=sk_test_...
  NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_...
  STRIPE_WEBHOOK_SECRET=whsec_...
  STRIPE_PRICE_ID=price_...
  UPSTASH_REDIS_REST_URL=https://...
  UPSTASH_REDIS_REST_TOKEN=...
  NEXT_PUBLIC_APP_URL=http://localhost:3000

Save and exit.

================================================================================
STEP 3: RUN THE APP (1 minute)
================================================================================

npm install
npm run dev

Open: http://localhost:3000

================================================================================
TROUBLESHOOTING
================================================================================

API key error?
  → Check .env.local has all variables
  → Verify keys are copied exactly (no spaces)
  → Regenerate keys if uncertain

Can't generate emails?
  → Check Redis connection in .env.local
  → Verify GEMINI_API_KEY is valid
  → Check that email input is not empty

Stripe webhook not working?
  → Locally: Use "stripe listen" command
  → In production: Ensure URL is HTTPS and reachable
  → Check webhook signing secret is correct

Auth not working?
  → Verify Clerk publishable key
  → Check Clerk dashboard has correct allowed origins
  → Clear browser cookies and try again
